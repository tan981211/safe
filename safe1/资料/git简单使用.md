# git提交代码操作步骤

1.   *将新建的文件或者修改过的文件添加到索引库*

   `git add .`

2. *将本地分支保存到本地仓库*
   `git commit -m "提交的动作名"`

3. *拉取远程仓库并合并到当前分支*

   `git pull origin 分支名`
   `如（拉取远程master分支并合并到当前分支）：git pull origin master`

4. *从本地提交到服务器*

   `git push origin 分支名`

   `如（提交到远程master分支）：git push origin master`
